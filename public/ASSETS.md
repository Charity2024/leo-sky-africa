# Leo Sky Africa — Asset Directory

Place client-provided assets in the folders below. Components reference these paths via `src/content/`.

## `/public/logos/`
| File | Usage |
|------|-------|
| Partner logo SVGs and PNGs | Partners marquee, partners page |

## `/public/logo/`
| File | Usage |
|------|-------|
| `logo.svg` | Navbar, footer, loading screen |
| `logo-mark.svg` | Favicon source, compact placements |
| `og-image.png` | OpenGraph / Twitter card (1200×630) |

## `/public/videos/`
| File | Usage |
|------|-------|
| `hero.mp4` | Homepage hero background (autoplay, muted, loop) |
| `{pillar}-hero.mp4` | Optional pillar page hero videos |

## `/public/images/`
| File | Usage |
|------|-------|
| `pillar-education.png` | Space Education imagery |
| `pillar-astrotourism.png` | Astrotourism imagery |
| `pillar-innovation.png` | Space Innovation imagery |
| `about.png` | About / mission section |
| `event-*.png` | Event card thumbnails |

## `/public/partners/`
Deprecated — use `/public/logos/` for partner logos instead.

## `/public/events/`
Full-resolution event photos for the Events page gallery and detail views.

## `/public/gallery/`
Program and experience gallery images for pillar landing pages.
